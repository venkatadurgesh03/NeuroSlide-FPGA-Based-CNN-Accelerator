# Testbench Files
This folder contains simulation and verification files for the NeuroSlide FPGA CNN Accelerator project.

## Purpose
The testbench is used to verify functional correctness of the RTL design modules such as:
- Top module operation
- Convolution engine behavior
- Line buffer functionality
- Clock/reset response
- Dataflow validation

## Notes
Simulation performed using Vivado simulator / RTL simulation tools.
